### React router-dom-v6

1. Router-dom NavLink 태그를 통해서
   <NavLink to={`/${item.name}`} onClick={()=>{}} className={({ isActive })=>{ return isActive ? activeLink : normalLink }}>

isActive라는 state를 받아와 버튼이 눌러져 활성화가 되어 있는 상태 true 와 false 의 스타일을 지정 할 수 있다.
